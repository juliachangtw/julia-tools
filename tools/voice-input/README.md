# 🎙️ 語音輸入工具（Windows）

在任何應用程式中，按快捷鍵說話，自動轉錄成文字並貼入游標所在位置。  
使用本機 **Whisper** 模型，完全離線、免費、支援中英文混打。

---

## ✨ 功能

| 功能         | 說明                                           |
| ------------ | ---------------------------------------------- |
| 全域快捷鍵   | 在任何應用（VS Code、Notion、Chrome…）都能啟動 |
| 本機 Whisper | 不需網路、不需 API Key、資料不上雲             |
| 自動偵測語言 | 中文、英文、中英混合皆可                       |
| 即時貼入     | 轉錄完成後自動貼入目前游標位置                 |
| 浮動提示視窗 | 右下角顯示狀態，不遮擋作業                     |

---

## 📦 系統需求

- **Windows 10 / 11**
- **Python 3.9 以上**（[下載 Python](https://www.python.org/downloads/)）
- 麥克風

---

## 🚀 安裝步驟

1. 在本資料夾按兩下執行 **`install.bat`**
2. 等待套件安裝完成（約 1-2 分鐘）

> 若 pyaudio 安裝失敗，請改用：
>
> ```
> pip install pipwin
> pipwin install pyaudio
> ```

---

## ▶️ 使用方式

1. 執行 **`run.bat`** 啟動工具
2. 右下角出現「正在載入語音模型...」提示
   - 首次啟動會自動下載 Whisper 模型（約 244 MB），之後不需再下載
3. 出現「就緒」後，切換到你要輸入文字的應用程式
4. 按 **`Ctrl + Shift + Space`** 開始錄音（出現 🔴 錄音中）
5. 說話...
6. 再按一次 **`Ctrl + Shift + Space`** 停止（自動轉錄並貼入）

### 退出工具

在右下角浮動視窗上按滑鼠**右鍵** → 「退出語音輸入」

---

## ⚙️ 設定

開啟 `voice_input.py`，修改最上方的設定區：

```python
HOTKEY     = "ctrl+shift+space"  # 快捷鍵（可改為其他組合）
MODEL_SIZE = "small"             # 模型大小（見下表）
LANGUAGE   = None                # None=自動偵測 / "zh"=強制中文 / "en"=英文
```

### 模型大小比較

| 模型       | 檔案大小 | 速度 | 精確度 | 建議用途           |
| ---------- | -------- | ---- | ------ | ------------------ |
| `base`     | ~74 MB   | 最快 | 普通   | 測試用             |
| `small`    | ~244 MB  | 快   | 好     | **預設，日常使用** |
| `medium`   | ~769 MB  | 中   | 更好   | 重要內容           |
| `large-v3` | ~1.5 GB  | 慢   | 最佳   | 高精確度需求       |

## ❓ 常見問題

**Q: 快捷鍵沒有反應？**  
A: 若目標程式是以「系統管理員」身分執行（如某些 IDE），需要以管理員身分執行 `run.bat`。

**Q: 中文辨識不准？**  
A: 嘗試換成更大的模型（`medium` 或 `large-v3`），或在設定中加 `LANGUAGE = "zh"`。

**Q: 貼上的是舊的剪貼簿內容？**  
A: 轉錄後工具會立即更新剪貼簿，若遇此問題請稍等 0.2 秒再觀察。

**Q: 要怎麼開機自動啟動？**  
A: 在開始選單搜尋「工作排程器」，新增啟動時執行 `run.bat` 的任務。

## 🔒 隱私說明

- 語音資料**全程在本機處理**，不傳送至任何伺服器
- 無雲端儲存、無帳號登入
- 程式關閉後不保留任何語音資料
