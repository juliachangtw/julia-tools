🎙️ 語音輸入工具

在任何應用程式中按快捷鍵說話，自動轉錄並貼入游標位置。
本機 Whisper 模型，完全離線、免費、支援中英文混打。

💻
Windows 桌面應用程式
這個工具需要下載到你的電腦上執行，不是網頁工具。請依照下方步驟安裝。

✨ 功能特色

⌨️
全域快捷鍵
在任何應用（VS Code、Notion、Chrome…）皆可啟動
🔒
完全離線
不需網路、不需 API Key、資料不上雲
🇹🇼
繁中 + 英文
中英混打，自動加標點，去除語氣詞
📋
即時貼入
轉錄完成後自動貼入目前游標位置

📦 系統需求

  • ✅ Windows 10 / 11
  • ✅ Python 3.9 以上（下載 Python）
  • ✅ 麥克風

🚀 安裝步驟

  1. 1
    從 GitHub 下載或 clone 整個專案到你的電腦
  2. 2
    進入 tools/voice-input/ 資料夾，雙擊執行 install.bat（約 1-2 分鐘）
  3. 3
    安裝完成後，雙擊執行 run.bat

▶️ 使用方式

1. 啟動後右下角出現浮動視窗「正在載入語音模型...」

首次啟動會自動下載 Whisper 模型（約 244 MB），之後不需再下載

2. 出現「就緒」後，切換到你要輸入文字的應用程式

3. 按 Ctrl + Shift + Space 開始錄音

4. 說話...

5. 再按一次 Ctrl + Shift + Space，轉錄完成自動貼入

右鍵點浮動視窗 → 「退出語音輸入」可關閉工具

隱私說明
語音資料全程在本機處理，不傳送至任何伺服器。無雲端儲存、無帳號登入。
